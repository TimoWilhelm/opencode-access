Welcome to OpenCode Access

You've reached the OpenCode access gateway. To get started, follow these steps.

Early Access: This integration is in early access. Features and authentication flows may change as we iterate.
  • Install OpenCode
    New install — run the install script (recommended):
    curl -fsSL https://opencode.ai/install | bash

    Alternatively: npm install -g opencode-ai or bun install -g opencode-ai

    Homebrew: brew install anomalyco/tap/opencode. If you previously ran brew install opencode (without the tap), uninstall first: brew uninstall opencode && brew install anomalyco/tap/opencode

    Already installed? Upgrade via opencode upgrade, then confirm with opencode --version.

  • Authenticate via Terminal
    Run this command in your terminal:
    opencode auth login
  • Start Using OpenCode
    Once authenticated, you're ready to use all OpenCode features. See opencode.ai/docs/config for model and provider configuration.
